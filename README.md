Excel Data Analysis Portfolio
This repository contains Excel-based data analysis projects focused on cleaning datasets, analyzing business data, and building dashboards for decision-making.

Objective
To demonstrate how Excel can be used to transform raw data into meaningful business insights through analysis, visualization, and reporting.

Tools Used
Microsoft Excel
Pivot Tables & Pivot Charts
Power Query
Advanced Excel Formulas (XLOOKUP, SUMIFS, IF)
Data Visualization (Charts & Dashboards)
Basic Statistical Analysis

Featured Project
Housing Market Analysis
Cleaned and processed 50,000+ housing records
Removed missing values, invalid entries, and outliers
Analyzed pricing trends using Pivot Tables
Performed correlation analysis to identify price drivers
Created calculated metrics such as price per square foot


Skills Demonstrated
Data Cleaning & Preparation
Exploratory Data Analysis (EDA)
KPI & Trend Analysis
Business Insight Generation
Dashboard Reporting
Key Insight:
Square footage was the strongest predictor of house price (0.75 correlation), while bedrooms and property age had minimal impact.
